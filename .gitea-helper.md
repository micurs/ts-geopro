# Gitea Helper Script

Script for managing Gitea issues and PRs via REST API.

## Security Setup

The script reads your Gitea token from:
1. `GITEA_TOKEN` environment variable (if set)
2. tea CLI config at `~/Library/Application Support/tea/config.yml` (fallback)

**Never commit tokens to the repository.**

## Usage

```bash
# List issues
./gitea-helper.sh list           # Open issues
./gitea-helper.sh list all       # All issues
./gitea-helper.sh list closed    # Closed issues

# Create issue
./gitea-helper.sh create "Title" "Body text"

# Comment on issue
./gitea-helper.sh comment 24 "My comment"

# Close/reopen issue
./gitea-helper.sh close 24
./gitea-helper.sh reopen 24

# Create PR
./gitea-helper.sh pr "PR Title" "PR Body" "head-branch" "base-branch"
```

## Setup

Ensure tea CLI is configured:
```bash
tea login
```

Or set environment variable:
```bash
export GITEA_TOKEN="your-token-here"
```
